# HackerLogin Landing Page

HackerLogin 동아리의 공식 랜딩 페이지입니다. React와 TypeScript를 기반으로 제작되었으며, Vite를 사용하여 빌드됩니다.

## 프로젝트 소개

이 프로젝트는 HackerLogin 동아리의 활동과 정보를 소개하는 랜딩 페이지입니다. 주요 기능은 다음과 같습니다:

- 메인 페이지 (`/`): 동아리 소개 및 주요 활동 하이라이트
- 지원 페이지 (`/apply`): 동아리 지원 관련 정보 및 양식
- 활동 페이지 (`/activities`): 동아리의 상세 활동 내역

## 기술 스택

- React
- TypeScript
- Vite
- React Router
- Tailwind CSS

## 시작하기

1. 저장소 클론
```bash
git clone https://github.com/HackerLogin/LandingPage.git
cd LandingPage
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```

4. 빌드
```bash
npm run build
```

## 프로젝트 구조

```
src/
├── components/     # 재사용 가능한 컴포넌트
├── pages/         # 페이지 컴포넌트

```
